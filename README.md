# RealTimeRx DevSecOps Assignment

## Overview
This repository contains the base code for implementing a DevSecOps pipeline.

## Objective
Build a CI/CD pipeline with integrated security checks using Jenkins, Docker, and Git.

## Rules
- Do NOT modify app/app.py or tests/test_app.py without approval
- `.env` must never be committed
- Use GitFlow (feature → develop → main)
- All changes must go through Pull Requests

## Deliverables
- Complete Jenkins pipeline
- Dockerized deployment
- Security integration (SAST, dependency, image scan)
- Monitoring implementation

Refer to PAD document for full requirements.
